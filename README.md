# Dominoes AI Player
Dominoes game engine and AI players implemented in Haskell - so a pure functional design.

## Overview
Simulates full dominoes matches including game rules, correct five-and-threes scoring, and turn logic.
Includes both a simple baseline player and a strategic AI player that selects moves based on scoring potential and game state.

## Tech stack
Haskell

## Run
ghci DomsMatch.hs
